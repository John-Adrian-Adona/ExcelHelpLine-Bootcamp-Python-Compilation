## Requirements

- Python 3.x
- `pandas` library
- `numpy` library
- `bs4` library

## Installation

1. Install Python 3.x from [python.org](https://www.python.org/).
2. Install the required libraries using pip:

    ```sh
    pip install pandas numpy bs4
    ```
